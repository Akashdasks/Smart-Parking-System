# Smart Parking System 🚗🅿️

Smart Parking is a **full-stack web application** that simplifies parking management for users and owners. It features **real-time slot availability**, **AI-based recommendations**, and **online booking**, all in a user-friendly interface.

---

## Features

- ✅ Real-time parking slot availability  
- ✅ AI-based recommendations for nearest and optimal parking  
- ✅ Online booking and payment management  
- ✅ Map integration to locate parking spots easily  
- ✅ Separate interfaces for parking users and owners
